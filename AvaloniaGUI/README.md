# FF GUI (Avalonia UI)

GUI for the [FFcollection](https://github.com/TcPirate1/FFTCG_collection).

Margin (left, top, right, bottom).

## Build

Ensure you have a `.NET SDK` installed and run `dotnet run` in the project directory.
